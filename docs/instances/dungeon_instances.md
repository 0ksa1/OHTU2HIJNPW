# Echoes of the Keep - Dungeon instances

## Yleinen suunnitelma:

-   jokainen dungeon sisältää uniikin mekaanikan, johon tarvitaan edellisestä dungeonista saadun abilityn käyttöä (poislukien ensimmäinen dungeon, jossa pelaajalla ei vielä ole lainkaana abilityä)
-   sisältää huoneita, puzzleja, ja lopussa boss/miniboss
-   jokaisessa dungeonissa hieman erilaisia vihollisia
-   pelissä tulee olemaan noin 4 eri pelattavaa dungeonia
-   dungeoneissa soi oma musiikki/ambient sound
-   jokaisen dungeonin lopussa korruptoitunut esine, joka pelaajan kuuluu kerätä. Korruptoituneet esineet avaavat uusia abilityjä
