# Ajatuksia

- Tyttö näyttää pelin teemaan kuulumattomalta, muistuttaa bloodbornesta Fire Keeper mutta pikselöity. Pelaajan ollessa selkeästi seikkailija ja tuleva NPC ritari saa tytön erottumaan selkeästi joukosta.
- Tarinan perusteella oletus, että tyttö olisi enemmänkin mummeli kuin nuori.
- Jatkossa dungeoneihin enemmän varietya -> ei käy silmille tylsäksi
- Liikkuminen ja hyökkäys intuiivista, mutta tutorial olisi hyvä lisä alkuun
- Tytölle ei yritetty edes puhua, sillä ei tiedetty että se on edes mahdollista
- Viholliset tuntuivat kuuluvan luonnollisesti luolastoon
