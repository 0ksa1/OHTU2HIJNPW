<h1 align="center">Echoes of the Keep</h1>

Echoes of the Keep is a 2D dark-fantasy RPG demo built with Godot 4.5 for UEF Software Engineering II course.  
The game features a central hub, four dungeons (boss/puzzle endings), and an ability-swapping system where only one ability can be equipped at a time.  
Explore new areas, meet new friends and foes, and enjoy the scenery while mastering new mechanics.

## Prerequisites

- Godot Engine 4.5
- Git

## Installation
